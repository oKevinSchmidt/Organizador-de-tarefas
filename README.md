# Organizador de Tarefas - C#

Projeto simples desenvolvido em C# para praticar:

- CRUD
- Listas
- Orientação a Objetos
- Estruturas de repetição
- Condicionais
- Manipulação de dados no console

## Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Remover tarefas
- Limpar lista
- Encerrar aplicação

## Tecnologias

- C#
- .NET
